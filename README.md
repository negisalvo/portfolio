# Professional Portfolio - Salvo Assefa

This is a modern, responsive professional portfolio website built with HTML5, CSS3, and JavaScript. It showcases my skills, projects, and professional background as a Full-Stack Developer and Mobile App Enthusiast.

## 🚀 Features

- **Responsive Design**: Optimized for all screen sizes (mobile, tablet, and desktop).
- **Modern UI**: Clean and professional aesthetic using Google Fonts and FontAwesome.
- **Project Showcase**: Detailed cards for highlighted projects with tech tags and links.
- **Smooth Navigation**: Intuitive smooth-scrolling between sections.
- **Contact Form**: Professional layout for potential inquiries.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Custom styling with variables, Flexbox, and Grid.
- **Google Fonts**: Poppins for clean typography.
- **FontAwesome**: Scalable vector icons.

## 📂 Structure

- `index.html`: Main structure and content.
- `style.css`: All custom styles and animations.
- `profile.jpg`: Professional profile image.
